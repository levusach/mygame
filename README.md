# The World

Терминальная C++ survival-игра с бесконечным процедурным миром.

## Сборка и запуск

```sh
make
./world
```

Если Unicode точно работает в терминале:

```sh
./world --unicode
```

Обычный `./world` специально использует ASCII-иконки, чтобы всё было видно даже в кривой локали.

Админская версия собирается отдельно:

```sh
make podvoh
./worldpodvoh
```

## Windows

Проект можно собрать в `world.exe` через MSYS2/MinGW:

```sh
pacman -S --needed mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-ncurses make
make win WIN_CXX=g++
./world.exe
```

Если собираешь кросс-компилятором из Linux:

```sh
make win
```

Админская Windows-сборка:

```sh
make win-podvoh
```

Для запуска на чужой Windows-машине рядом с `world.exe` могут понадобиться DLL из MSYS2/MinGW, в первую очередь `libncursesw*.dll`, `libwinpthread-1.dll`, `libgcc_s_seh-1.dll` и `libstdc++-6.dll`.

В репозитории есть GitHub Actions workflow `Windows build`. Он собирает Windows-версию, кладёт нужные DLL рядом с `world.exe`, делает архив `TheWorld-Windows.zip` и установщик `TheWorldSetup.exe`. При пуше тега `v*` эти файлы автоматически прикрепляются к GitHub Release.

## Цель на старте

1. Найди деревья `T` и камни `0`.
2. Сломай дерево или камень: `ЛКМ` по клетке или пробел по направлению взгляда.
3. Открой инвентарь на `I`.
4. Скрафти верстак: открой крафт на `C`, выбери рецепт и нажми `E`. Без верстака доступен только рецепт верстака.
5. Выбери верстак в хотбаре клавишами `1`-`9`.
6. Поставь верстак `ПКМ`.
7. Стой рядом с верстаком и открой крафт на `C`, чтобы делать продвинутые вещи.

## Миры И Сохранения

- Новая игра создаёт отдельный файл в папке `worlds/`.
- Каждый новый мир получает свой seed, поэтому озёра, леса, камень, пшеница и руда раскладываются иначе.
- Старый одиночный `world.sav` всё ещё виден в меню загрузки как `old world.sav`.
- `C - continue` в главном меню открывает список миров.

## Админ Версия

Обычный `world` не содержит админ-команд. Для тестов и строительства есть `worldpodvoh`: та же игра, но slash-команды в чате выполняются от имени хоста.

Чат открывается на `T`. Обычные сообщения просто появляются в истории. Команды начинаются с `/`.

Команды хоста:

- `/help`: показать подсказку.
- `/items`: показать id предметов.
- `/give <id> [amount]`: выдать предмет. Если количество не указано, выдаётся `999`.
- `/tp <x> <y>`: телепорт на любые координаты.
- `/pos1 here` / `/pos2 here`: поставить точки worldedit по текущей позиции игрока.
- `/pos1 <x> <y>` / `/pos2 <x> <y>`: поставить точки worldedit координатами.
- `/blocks`: показать символы блоков.
- `/fill <tile>`: заполнить прямоугольник между `pos1` и `pos2` блоком.

В будущей локальной мультиплеерной версии эти команды должны приниматься только от хоста.

## Управление

- `WASD` или стрелки: ходьба.
- Две стрелки/WASD одновременно: диагональ, насколько это позволяет терминал.
- Мышь: направление взгляда.
- `ЛКМ`: ударить/сломать в радиусе 3 блоков.
- `ПКМ`: поставить выбранный блок перед игроком.
- Пробел: ударить по направлению взгляда.
- `1`-`9`: выбрать слот хотбара.
- `I`: открыть/закрыть инвентарь.
- `E`: использовать контекстное действие, открыть сундук рядом, взять/положить стак в инвентаре.
- `C`: открыть/закрыть отдельное меню крафта.
- `T`: открыть чат.
- `Shift+E` в меню крафта: сделать выбранный рецепт столько раз, сколько хватает ресурсов.
- `q`: выход.

## Мир

- `@`: игрок.
- `.`: пустое место.
- `0`: камень, непроходимый.
- `~`: вода, непроходимая.
- `:`: пляж.
- `T`: дерево, через него можно пройти.
- `&`: корова.
- `#`: верстак.
- `F`: печка.
- `D`: дверь.
- `C`: сундук.
- В режиме `--unicode` они могут отображаться как `牛`, `⚒`, `♨`, `▥`.

## Инвентарь

- Первые 9 ячеек инвентаря являются хотбаром.
- `1`-`9` выбирают активный слот.
- В инвентаре `WASD`/стрелки двигают курсор.
- `E`/`Enter` в инвентаре берёт, кладёт, объединяет или меняет предметы.
- `C` открывает список доступных рецептов. `E` крафтит один раз, `Shift+E` крафтит всё возможное.
- Сундук открывается на `E`, если он стоит рядом или перед игроком. Курсор ходит по слотам инвентаря и сундука, `E` переносит стак.

## Ресурсы

- Камень `0` даёт `Stone`.
- Дерево `T` даёт `Wood`.
- Корова `&` даёт `Raw Meat`.
- У воды `~` можно набрать `Dirty Water`.

## Крафт

Без верстака:

- `Workbench`: 4 `Wood`.

Рядом с верстаком:

- `Furnace`: 8 `Stone`.
- `Door`: 3 `Wood`.
- `Chest`: 8 `Wood`.
- `Spear`: 1 `Wood` + 1 `Stone`.

Крафт делается в отдельном меню на `C`.

## Еда И Вода

- Сырое мясо есть нельзя.
- Грязную воду пить нельзя.
- Рядом с печкой `E` жарит `Raw Meat` в `Cooked Meat`, кипятит `Dirty Water` в `Boiled Water` или плавит руду.
- Если выбран `Cooked Meat`, `E` ест его и восстанавливает голод.
- Если выбрана `Boiled Water`, `E` пьёт её и восстанавливает жажду.

## Копьё

- Скрафти `Spear` рядом с верстаком.
- Выбери копьё в хотбаре.
- Удар `ЛКМ` или пробел сначала пробует использовать копьё.
- Копьё летит по направлению взгляда и может убить корову на расстоянии.

## HUD

Сверху слева показываются:

- текущий предмет;
- количество;
- сердца;
- голод;
- жажда;
- день или ночь.

## Если Мышь Или Unicode Глючат

- Запускай в нормальном терминале с mouse tracking: Kitty, Alacritty, GNOME Terminal, Konsole.
- Проверь UTF-8 локаль:

```sh
locale
```

- Желательно, чтобы было что-то вроде `LANG=en_US.UTF-8` или `LANG=ru_RU.UTF-8`.
- Если Unicode всё равно кривой, проблема почти всегда в шрифте терминала или локали, а не в `sudo`.
